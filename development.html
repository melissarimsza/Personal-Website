Edges2Kirby

Edges2Kirby is a machine learning pased web project. It uses a custome model applied with tensorflow.js to color fill a user drawn kirby. Check out the project HERE!


2048

2048 is a desktop number-based puzzle game. Combine numbers on the game board to make matches and work your way up through powers of two. Win the game by combinging enough numbers to reach 2048!



SDFC Wordle

Based on the sensation Wordle (owned by the New York Times), SDFC Wordle is a custom version created for the Arizona State Sun Devil Fitness Center. It features health and wellness based words and ASU branding. Check out the project HERE!


Hotdog/Not Hotdog

Based off of the app "Seefood" from the TV show Silicon Valley. This goofy app combines the power of the Apple MobileNetV2 image classifier with the ease and functionality of SwiftUI to create a hotdog identifying app.
